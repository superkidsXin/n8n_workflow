# n8n-nodes-aliyunoss
Aliyun OSS upload node for n8n.
## Features
- Upload binary data to OSS
## Setup
1) Install dependencies: `npm install`
2) Build: `npm run build`
3) In n8n, install this community node package and configure credentials `Aliyun OSS API`.
## Node
- **Aliyun OSS**: Upload operation reads `Binary Property` (default `data`) and uploads to `Bucket`/`Object Key`.
